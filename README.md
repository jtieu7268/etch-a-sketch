# etch-a-sketch

A browser version of an Etch-A-Sketch. 

Color in squares by hovering over them. Create a new grid by pressing the "NEW GRID" button and setting the size of the grid (the number of squares per side).

## Usage

Access the web app here [here](https://jtieu7268.github.io/etch-a-sketch/).

## Technologies

Implemented using HTML, CSS, and Javascript. Deployed using GitHub pages.
